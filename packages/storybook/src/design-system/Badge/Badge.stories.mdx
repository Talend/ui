## Interactions

### With Operator (Operator = true)

When Operator is true, clicking on Operator button show a dropdown with several operators to choose one.

### With Dropdown (Value = Single ; Dropdown = true)

When Dropdown is true and Value is set to "Single", clicking on the value part show a list of choice to select one.

When done, chosen value is displayed in the value part.

### With Popover (Value = Multi ; Dropdown = true)

When Dropdown is true and Value is set to "Multi", clicking on one value in values part show a popover displaying whatever you want.

### With Close button (OnClose = _function_)

When OnClose is provided, clicking on close button call OnClose handler.
