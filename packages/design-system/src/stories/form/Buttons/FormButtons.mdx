## Content

### Buttons order

Sort the buttons from the least to the most important.

- The main action button is placed on the right
- Do not use more than one primary button per form
- Other form buttons should be secondary buttons
- Depending on the context, consider using a back or cancel button. This button should always be placed at the very left.
- The other action buttons are placed on the right side.

### Buttons with icons

Consider using icons only for the main action button when its label isn't understandable enough.

## Interactions

The buttons related to the form are part of a **sticky footer**.

The footer containing the buttons should stick to the screen bottom once the user starts scrolling down.

**This behaviour is not implemented by `Form.Buttons`.**
