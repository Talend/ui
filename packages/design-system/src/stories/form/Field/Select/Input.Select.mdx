## Content

When you use clear terms and order them logically in the list, you enable users to scan quickly and select what they need.

<Use>
	<Use.Do>
		<ul>
			<li>Aim for terms and words which are very familiar to users.</li>
			<li>
				Write items/option in order (alphabetical order or any other order which makes sense).
			</li>
		</ul>
	</Use.Do>
	<Use.Dont>
		<ul>
			<li>
				Donâ€™t use vague or misleading options that are difficult to understand by average users.
			</li>
		</ul>
	</Use.Dont>
</Use>

## Interactions

- When a user clicks into the text input field, they trigger the selection menu which is the browser default one (for compatibility reason).

## Accessibility

The keyboard gestures are the default ones, such as:

- Press arrows keys to display and navigate into the menu.
- Press <kbd>ENTER</kbd> or <kbd>SPACE</kbd> to select the option.
