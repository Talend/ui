# Radio

`Form.Radio` inputs are used when a question has only one answer. Radio only lets users select one option from a set.

Use radio buttons for exclusive selection when users need to see all available options side-by-side.

Otherwise, consider using [checkboxes](components-form-form-fields-checkbox--checkbox-fill-states) or a [select field](components-form-form-fields-select).

## Content

Radio buttons appear as a list of two or more options where users can select only one option. It's important to make their labels comprehensive and clearly distinct.

<Use>
	<Use.Do>
		<ul>
			<li>Use positive and active wording for labels.</li>
			<li>
				Aim for clear and distinct labels to ease the reading flow and help users make the right
				choice.
			</li>
			<li>Start each label with capital letter.</li>
		</ul>
	</Use.Do>
	<Use.Dont>
		<ul>
			<li>
				Don’t use vague or misleading labels that are difficult to understand by average users.
			</li>
			<li>Don’t use punctuations at the end of labels.</li>
		</ul>
	</Use.Dont>
</Use>

## Interactions

- The label must always be clickable to check or uncheck.

## Accessibility

Press <kbd>Tab</kbd> to focus on a radio button.

Press <kbd>Tab</kbd> or <kbd>Shift</kbd> + <kbd>Tab</kbd> to navigate between radio buttons.

Press the arrows keys to change the selection.
