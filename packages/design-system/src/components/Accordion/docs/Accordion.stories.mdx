### Sizes

The accordion component exists in two sizes: `Default` and `Small`

Consider using a `Small` accordion when it is embedded into another component such as a Drawer.
If the accordion is used full width into the layout then use the `Default` size.

## Content

<Use>
	<Use.Do>
		<ul>
			<li>Use sentence case in section headings.</li>
			<li>
				Try to make your headings as short as possible while keeping them clear and descriptive.
			</li>
		</ul>
	</Use.Do>
	<Use.Dont>
		<ul>
			<li>Don't use title case or all caps in accordion headings (except for acronyms).</li>
		</ul>
	</Use.Dont>
</Use>

## Usage

### This is an Accordion

When multiple accordions are stacked only one panel can be opened at once. If you need to open more than one accordion at the same time, then use a Collapsible panel.

<FigmaImage
	src="https://www.figma.com/file/CDfr4jLz1m6Ud2RNi4qpQJ/Accordion?node-id=1159%3A45365"
	alt="accordion opened"
/>

### This is a Collapsible panel

A Collapsible panel allows the user to open all the panel if needed at the same time.

<FigmaImage
	src="https://www.figma.com/file/CDfr4jLz1m6Ud2RNi4qpQJ/Accordion?node-id=1159%3A45513"

## Accessibility

- Press <kbd>ARROW UP</kbd> or <kbd>ARROW DOWN</kbd> to expand or collapse the previous or next accordion sections.
