# Message

Use Message when information needs to be displayed in a single context (such as a notification center, or a list of warnings etc...).

Messages have one of these semantic values: informative, success, warning or error.

Its core text content is mandatory. Other elements are optional. The hyperlink element is only available on the Message, never on MessageCollection.

## MessageCollection

Use MessageCollection when multiple messages about the same event happens. It can be informative, success, warning or error.

Title, Text and MainAction are mandatory elements.

Only one “MainAction” is available. All other actions must be in a dropdown. This is true for collections and regular messages.

Decorative elements in the background do not count in the Message’s box size. Use absolute elements or shadows to replicate.
