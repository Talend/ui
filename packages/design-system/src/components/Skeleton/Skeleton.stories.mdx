## Interaction

Skeletons use the heartbeat animation.
The cursor is set to `progress`.

## Accessibility

They are unitary hidden for screen readers by default.
The goal is to set the current loading content to `aria-busy="true"` instead.
Here is an article about [more accessible skeletons](https://adrianroselli.com/2020/11/more-accessible-skeletons.html) written by Adrian Roselli.
