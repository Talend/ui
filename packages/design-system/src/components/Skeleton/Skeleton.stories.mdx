## Interaction

Skeletons use the heartbeat animation.
The cursor is set to `progress`.

## Accessibility

Each Skeleton component is hidden to screen readers by default.
The goal is to set the current loading content to `aria-busy="true"` instead.
Here is an article about [more accessible skeletons](https://adrianroselli.com/2020/11/more-accessible-skeletons.html) written by Adrian Roselli.

## Usage

As you've seen above, `Skeleton` can be used by importing the single variant you need (ie: `SkeletonButton` or `SkeletonHeading`).

You can also use `Skeleton` and specify the variant you need to use: its props will then match the variant's own.
