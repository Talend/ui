## Content

### Buttons order

Sort the buttons from the less to the more important.

- The main action button is placed on the right
- Avoid to use more than one primary button per form
- Other form buttons should be secondary buttons
- Depending on the context, consider using a back or cancel button. This button should always be placed at the very left.
- The other action buttons are placed on the right side.

### Buttons with icons

Consider using Buttons with icons only for the main action when the label isn't enough understable.

## Interactions

The buttons related to the form are part of a **sticky footer**

The footer containing the buttons will be stick to the screen bottom once the user starts scrolling down.
