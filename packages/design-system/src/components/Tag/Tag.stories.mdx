## Content

Tags label UI items with keywords that help organize and categorize the items or draw more attention to them.

<Use>
	<Use.Do>
		<ul>
			<li>Use sentence case in tag labels.</li>
			<li>Use short labels on tags for easy scanning.</li>
			<li>Use clear and concise text.</li>
			<li>Use adjectives or nouns.</li>
		</ul>
	</Use.Do>
	<Use.Dont>
		<ul>
			<li>Don't use verbs.</li>
			<li>Don't use all caps in labels (except for acronyms).</li>
		</ul>
	</Use.Dont>
</Use>
