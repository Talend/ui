## Content

By default, each Status variants has a label:

- In progress for `Status.InProgress`
- Successful for `Status.Successful`
- Failed for `Status.Failed`
- Warning for `Status.Warning`
- Canceled for `Status.Canceled`

Depending on the use case, it is possible to have another label instead but keeping the same meaning.

<Canvas>
	<Story story={Stories.CustomSuccessful} />
	<Story story={Stories.CustomInProgressIcon} />
</Canvas>

## Usage

<Canvas>
	<Story
		name="usage"
		args={{
			variant: 'Successful',
			hideText: false,
			children: 'Done',
		}}
		argTypes={{
			variant: {
				description: 'Status variation',
				control: {
					type: 'select',
					options: [''].concat(Object.keys(Status).filter(key => /^[A-Z]/.test(key))),
				},
			},
			hideText: {
				description: 'If Status should hide text or not',
				control: {
					type: 'boolean',
				},
			},
			children: {
				description: 'Status text',
				control: {
					type: 'text',
				},
			},
		}}
	>
		{({ variant, ...rest }) => {
			const Component = Status[variant] || Status;
			Component.displayName = `Status${variant && `.${variant}`}`;
			return <Component {...rest} />;
		}}
	</Story>
</Canvas>
