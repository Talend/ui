## Content

By default, each Status variants has a label:

- In progress for `StatusInProgress`
- Successful for `StatusSuccessful`
- Failed for `StatusFailed`
- Warning for `StatusWarning`
- Canceled for `StatusCanceled`

Depending on the use case, it is possible to have another label instead but keeping the same meaning.

## Usage

<Canvas>
	<Story
		name="usage"
		args={{
			variant: 'warning',
			hideText: false,
			children: 'Done',
		}}
		argTypes={{
			variant: {
				description: 'Status variation',
				control: {
					type: 'select',
					options: [...possibleVariants],
				},
			},
			hideText: {
				description: 'If Status should hide text or not',
				control: {
					type: 'boolean',
				},
			},
			children: {
				description: 'Status text',
				control: {
					type: 'text',
				},
			},
		}}
	>
		{({ variant, ...rest }) => {
			const Component = StatusPrimitive;
			if (variant === 'warning') {
				const Component = StatusWarning;
				Component.displayName = 'StatusWarning';
				return <Component {...rest} />;
			}
			if (variant === 'successful') {
				const Component = StatusSuccessful;
				Component.displayName = 'StatusSuccessful';
				return <Component {...rest} />;
			}
			if (variant === 'failed') {
				const Component = StatusFailed;
				Component.displayName = 'StatusFailed';
				return <Component {...rest} />;
			}
			if (variant === 'inProgress') {
				const Component = StatusInProgress;
				Component.displayName = 'StatusInProgress';
				return <Component {...rest} />;
			}
			if (variant === 'canceled') {
				const Component = StatusCanceled;
				Component.displayName = 'StatusCanceled';
				return <Component {...rest} />;
			}
			return <Component {...rest} />;
		}}
	</Story>
</Canvas>
