## Content

By default, each Status variants has a label:

- In progress for `StatusInProgress`
- Successful for `StatusSuccessful`
- Failed for `StatusFailed`
- Warning for `StatusWarning`
- Canceled for `StatusCanceled`

Depending on the use case, it is possible to have another label instead but keeping the same meaning.

	<Story story={Stories.CustomSuccessful} />

## Usage

You can directly use the presets `StatusInProgress` etc... but in some cases you may want to handle the variant through a prop.

You can.

<Canvas>
	<Story
		name="usage"
		args={{
			variant: 'warning',
			hideText: false,
			children: 'Done',
		}}
		argTypes={{
			variant: {
				description: 'Status variation',
				control: {
					type: 'select',
					options: Object.values(variants),
				},
			},
			hideText: {
				description: 'If Status should hide text or not',
				control: {
					type: 'boolean',
				},
			},
			children: {
				description: 'Status text',
				control: {
					type: 'text',
				},
			},
		}}
	>
		{({ variant, ...rest }) => {
			const Component = Status;
			Component.displayName = 'Status';
			return <Component variant={variant} {...rest} />;
		}}
	</Story>
</Canvas>
