# Getting Started

## For designers

Check it out on [Figma](https://www.figma.com/file/hMYM9HGXajJpWdGwRb5ITR/Coral).

## For developers

Use the package manager you want to install the package `@talend/design-system` and its peer dependencies.

```sh
npm i -S react react-dom react-is classnames @talend/design-system
```

or

```sh
yarn add react react-dom react-is classnames @talend/design-system
```
