## States

When the link is disabled the whole component is displayed with 0.54 opacity

## Content

Use a meaningful link text that indicates what will happen when the user interacts with the link.

## Accessibility

Press <kbd>TABS</kbd> to focus on a link

Press <kbd>ENTER</kbd> to open a link

## Usage

When a link opens the page in a new tab, please use the external link icon right after.
