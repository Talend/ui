## Style

### Sizes

| Property                       | Value             | 
| ------------------------------ | ----------------- | 
| icon           	             | 1.2rem            |

### Spaces

| Property          		                | Value             | 
| ----------------------------------------- | ----------------- | 
| (if icon) padding-right & padding-left	| $smaller          | 

### Colors

Icons should always be the same color as the link text.

| Property                       | Value                               | 
| ------------------------------ | ----------------------------------- | 
| font & icon		             | $lochmara                           |
| font & icon	(hover)	         | shade($lochmara, 20%)              |
| font & icon	(active)         | shade($lochmara, 40%)              |

### Font size

| Property                       | Value             | 
| ------------------------------ | ----------------- |
| text							 | 1.4rem 			 |

### Font weight

| Property                       | Value             | 
| ------------------------------ | ----------------- |
| text							 | semibold 		 |

### Text transform

| Property                       | Value             | 
| ------------------------------ | ----------------- |
| text (hover & active)			 | underline 		 |

### Disabled

When the link is disabled the whole component is displayed with 0.54 opacity.

<Preview>
    <Story name="disabled">

	</Story>
</Preview>

### External

When the link targets a new domain, we add an external-link icon right after.

<Preview>
    <Story name="external">
        <Link href="https://www.talend.com" target="_blank">talend.com</Link>
	</Story>
 </Preview>

The attribute `rel="noopener noreferrer"` is added if its target is `_blank`.

## Content

Use a meaningful link text that indicates what will happen when the user interacts with the link.
