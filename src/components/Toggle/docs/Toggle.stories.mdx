## Content
A toggle is an on/off switch which is used to change the state of system functionalities and preferences.

<Use>
	<Use.Do>
  <ul>
<li>Keep labels for toggles short and direct.</li>
<li>Use the label to describe what the control will do when the switch is on.</li>
  </Use.Do>
	<Use.Dont>
  <ul>
<li>Don't use neutral or ambiguous label.</li>

  </ul>
  </Use.Dont>
</Use>
