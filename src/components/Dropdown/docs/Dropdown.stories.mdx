## Content

**Keep menu items to a single line of text as much as possible**

If not, when the label is too long for the available horizontal space, they wrap to form another line.

## Accessibility

Classic behaviors expected:

- Press <kbd>TABS</kbd> to focus on the dropdown

- Press <kbd>TABS</kbd> to navigate in the dropdown menu

- Press <kbd>SPACE</kbd> or <kbd>ENTER</kbd> to activate the dropdown menu item

For more information, please go to accessibility page

## Usage

**Max height**

The max-height of a dropdown is 32rem. A scroll-bar has to be added after exceeding 32rem longer items.
This rule is not applicable for the header-bar dropdowns.
