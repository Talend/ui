# Accordion

The accordion component allows the user to display section of content; each item can be collapsed or expanded.

### Sizes

| Attribute  | Variable |
| ---------- | -------- |
| max-height | 37rem    |
| min-width  | 25rem    |

The max-height of a accordion item is 37rem. A scroll-bar has to be added in the content after exceeding 37rem longer items.

</Use>

## Interaction

## Accessibility

- Press <kbd>ENTER</kbd> or <kbd>SPACE</kbd> to expand or collapse the accordion when the area is focused.

## Usage

The accordion component can be placed as main page content or placed inside the container such as a drawer or an overview tile.
