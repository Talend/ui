## Sizes <a id="sizes" />

Cobalt support two buttons sizes: **default** and **small**.

### Default

| Attribute                      | Variable      |
| ------------------------------ | ------------- |
| height                         | 3.4rem        |
| padding-left; padding-right    | \$large       |
| border                         | solid, 0.1rem |
| border-radius                  | rectRadius    |
| _ (if icon) _ icon-size        | 1.4rem        |
| _ (if icon) _ margin-right     | \$smaller     |
| _ (for dropdown) _ icon-size   | 0.8rem        |
| _ (for dropdown) _ margin-left | \$smaller     |

### Small

The small size is available for primary, secondary and ghost buttons â€” but shouldn't be use for buttons with icons or dropdown buttons.

| Attribute                   | Variable      |
| --------------------------- | ------------- |
| height                      | 2.8rem        |
| padding-left; padding-right | \$small       |
| border                      | solid, 0.1rem |
| border-radius               | rectRadius    |

### Icon buttons

Icon buttons has two sizes: **large** and **small**. They can also be used as dropdown but without the caret icon.

| Large                | Variable      | Small         | Variable      |
| -------------------- | ------------- | ------------- | ------------- |
| icon-size            | 2.4 rem       | icon-size     | 1.6rem        |
| **If circle border** |
| size                 | 2.4 rem       | size          | 1.6rem        |
| border               | solid, 0.2rem | border        | solid, 0.2rem |
| icon-size            | solid, 1.2rem | icon-size     | solid, 0.8rem |
| border-radius        | rectCircle    | border-radius | rectCircle    |

## Content

- Except in some very specific situations, the button must contain a maximum of 3 words action

- Use sentence-style capitalization (only the first word in a phrase and any proper nouns capitalized)

## Interaction

- **Buttons should all have a tooltip on hover displaying the description**

<Row>
	<Col>
		<Story id="components-tooltip--basic" />
	</Col>
</Row>

- **Buttons has a loading states**

When the user clicks on the button, a spinner will appear on the left of the label. No actions will be possible until it's loading.
In case of button with icon inside, the spinner will replace the icon as long as it's loading.

## Accessibility

Classic behaviors expected:

- Press <kbd>TABS</kbd> to focus on the button

- Press <kbd>SPACE</kbd> or <kbd>ENTER</kbd> to activate the button

For more information, please go to accessibility page

## Usage

### Buttons order

<img src={btnOrder} alt="button order in a page" className="full" class="full" />
