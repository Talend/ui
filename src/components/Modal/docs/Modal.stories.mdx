### Where to place buttons?
Buttons are always at the bottom of the dialog box. In case of scroll bar into the body, they should be fixed at the bottom.
Dismiss actions should be on the left  / Confirmation actions on the right

### Prioritize actions
One main action should be expected from the user emphasized by the use of a primary button. Secondary buttons can be used for second level actions. 2 primary validation/action buttons cannot be used

### Cancel or close?
In Confirmation dialog box, close icon cannot be displayed. 
A Cancel button shall be displayed on the left side. Closing icon can be used only for informative dialog box.
