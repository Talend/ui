# Radio

Radio buttons are used when a question has only one answer. Radio buttons allow the user to select one option from a set. 
Use radio buttons for exclusive selection if you think that the user needs to see all available options side-by-side.

Otherwise, consider checkboxes, toggle switch or a dropdown.

## Content

Radio buttons appear as a list of two or more options on UI where users can select only one option, so it's important to make their labels comprehensive and clearly distinct.

<Use>
	<Use.Do>
		<ul>
			<li>Use positive and active wording for lables.</li>
			<li>Aim for clear and distinct lables to ease the reading flow and help users make the right choice.</li>
			<li>Start each label with capital letter.</li>
		</ul>
	</Use.Do>
	<Use.Dont>
		<ul>
			<li>Don’t use vague or misleading labels that are difficult to understand by average users.</li>
			<li>Don’t use punctuations at the end of labels.</li>
		</ul>
	</Use.Dont>
</Use>


## Interaction

- If a label is provided, it should always be clickable to check or uncheck.
- A set of radio buttons should have by default one option selected.

## Accessibility

Press <kbd>TABS</kbd> to focus on a radio button.

Press <kbd>TABS</kbd> or <kbd>Shift TABS</kbd> to navigate between radio buttons.

Press the <kbd>ENTER</kbd> or <kbd>SPACE</kbd> to toggle the radio button between selected and not selected.
