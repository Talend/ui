# Radio

Radio buttons are used when a question has only one answer. Radio buttons allow the user to select one option from a set. 
Use radio buttons for exclusive selection if you think that the user needs to see all available options side-by-side.

Otherwise, consider checkboxes, toggle switch or a dropdown.

## Accessibility

Press <kbd>TABS</kbd> to focus on a radio button.

Press <kbd>TABS</kbd> or <kbd>Shift TABS</kbd> to navigate between radio buttons.

Press the <kbd>ENTER</kbd> or <kbd>SPACE</kbd> to toggle the radio button between selected and not selected.
