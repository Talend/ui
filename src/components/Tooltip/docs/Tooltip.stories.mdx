# Tooltip

Tooltip appears on mouse hover in order to identify or define relevant information about elements.

## Content

- Tooltip should not include clickable content.
- The tooltip content shouldn't be longer than 3 or 4 lines to make it easy to read. If you need more space to display the message please use a popover.
- Tooltip should not display icons or images, only text.
