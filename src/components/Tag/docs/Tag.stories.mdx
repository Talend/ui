# Tag

The Tag component is useful to emphasize information to the user, works best with single word values.

## Content

Tags label items with keywords that help organize and categorize the items or draw more attention to them.

<Use>
	<Use.Do>
		<ul>
			<li>Use short labels on tags for easy scanning.</li>	
			<li>Use clear and consise text.</li>
			<li>Use adjectives or nouns, not verbs.</li>
		</ul>
	</Use.Do>
	<Use.Dont>
		<ul>
			<li>Don't use all caps in labels.</li>
		</ul>
	</Use.Dont>
</Use>
