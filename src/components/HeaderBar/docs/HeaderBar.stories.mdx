# Header Bar

The header bar is the place, where the user finds useful information and tools to for the application.
The header is placed on the top and takes the full window width.

### App switcher

The App switcher displays ALL applications that the user has access to. No entry in the app list should be disabled. Entries that cannot be used (e.g. no permissions) shouldn't be displayed at all (no disabled state). 
There is no selected state, from the same app user can select the app again and the order of the application names remain always the same.

### In-product chat

By clicking on the chat button at any time, a chat widget is displayed allowing users to send messages to the Talend team to provide feedback or questions.

### User  Menu

All versions of the product header bar give access to the user menu.  The content vary depending on the application and the application context (on premise)

### Product specific actions

Optionally, products may add specific actions as buttons, dropdown or links.
