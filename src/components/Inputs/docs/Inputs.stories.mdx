# Inputs

Text fields can display user input in the following ways:
- Single line text fields display only one line of text
- Text areas are fixed-width fields
Use text fields in forms to help people enter, select, and search for text. 
Text fields are usually found within a form but can also be part of a modal, search, or widgetâ€¦ 
Common text input types include: usernames, descriptions, URLs, emails, addresses, and plain text searches.
