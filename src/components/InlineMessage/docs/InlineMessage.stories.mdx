# Inline message

Inline message highlights information necessary to display for the user in many different contexts. It can be additional information related to system status, it can be a required action to complete the current task.

## Content <a id="content" />

- Be clear and conversational
- Use direct language

## Interaction <a id="interaction" />

Inline messages are non clickable, except if there is a link. In that case, only the link will be clickable. 
Please refer to the hyperlink guidelines for more information.
