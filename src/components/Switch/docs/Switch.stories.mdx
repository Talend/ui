# Switch

Switches are a quick way to make a selection between two states (or three sometimes)  or on-off state instantly. There is always a default selection for switches.

## Content

Use switches to change the state of system functionalities and preferences on or off, or to select among more than two states. Labels for switches should give users a strong indication of what they are selecting.

<Use>
	<Use.Do>
		<ul>
			<li>Keep labels for switches short and direct.</li>
			<li>For on/off switches, use the label to describe what the control will do when the switch is on.</li>
			<li>For switches to select among more options, use short clear lables.</li>
		</ul>
	</Use.Do>
	<Use.Dont>
		<ul>
			<li>Don't use neutral or ambiguous labels.</li>
		</ul>
	</Use.Dont>
</Use>
