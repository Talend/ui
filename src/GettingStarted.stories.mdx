# Getting Started

## For designers

Check it out on [Figma](https://www.figma.com/files/915288686127639393).

## For developers

```sh
npm i -S react react-dom react-i18next styled-components @talend/design-system
```

Or, use CDN versions:

```html
<!doctype html>
<html>
	<head>
		<meta charset="utf-8" />
		<title>Talend Design System example</title>
		<meta name="viewport" content="width=device-width, initial-scale=1">
	</head>
	<body>
		<div id="root"></div>
		<script crossorigin src="https://unpkg.com/react@16/umd/react.production.min.js"></script>
		<script
			crossorigin
			src="https://unpkg.com/react-dom@16/umd/react-dom.production.min.js"
		></script>
		<script
			crossorigin
			src="https://unpkg.com/styled-components/dist/styled-components.min.js"
		></script>
		<script crossorigin src="https://statics.cloud.talend.com/@talend/design-system/next/dist/TalendDesignSystem.min.js"></script>
		<script>
			const { Button, ThemeProvider } = TalendDesignSystem;
			const App = React.createElement(
				ThemeProvider,
				null,
				React.createElement(Button, {}, 'Example'),
			);
			ReactDOM.render(App, document.getElementById('root'));
		</script>
	</body>
</html>
```
